Magic Layout Tool is only functional under Linux OS.

After installing cygwin, relocate the cygwin file directly to C: Repository and open Cygwin Terminal.

Open One Terminal and extract zip file
(make sure magic.tgz, tcltk.tgz are inside the new relocated cygwin file)

tar xfz magic-7.5.78.tgz

tar xfz tcltk_x11_win.tgz

and copy,

Xwin -multiwindow

Open a new Terminal,

export PATH=$PATH:/usr/local/lib

export DISPLAY=:0

magic
